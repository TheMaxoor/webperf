# Accessing website

Base URL: https://webperf.mrossignol.fr

Branches URLs : https://\<branch-name>.mrossignol.fr

Branches are automatically deployed on push event.

## Commands

Install dependencies :

```bash
npm install
```

Build :

```bash
npm run build
```

```bash
npm run build:dev
npm run build:prod # For minification
```

Local dev start :

```bash
npm run develop
```

Project start :

```bash
npm run start
```
